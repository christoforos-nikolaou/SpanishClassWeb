### Notas de gramática

### Ser y Estar

Aqui tienes una tabla que resume las diferencias entre "ser" y "estar":

| Uso                      | Ser                                                                 | Estar                                                                                                          |
|--------------------------|---------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------|
| Identificación           | Para identificar a alguien o algo.                      | No aplica directamente para identificación.                                                         |
| Características          | Características permanentes o situación permanente.      | Características temporales o estados emocionales/físicos temporales.                               |
| Enfatizar estado         | Estado civil o característica permanente.                | Estado temporal o comparación con situación habitual.                                               |
| Participios              | No aplica.                                              | Indica resultado de una acción o acontecimiento (ej. está roto).                                   |
| Lugar, hora, fecha       | Indica lugar, hora o fecha de un acontecimiento.         | Indica situación física o temporal (ej. dónde estabas anoche).                                     |
| Expresiones y locuciones | No aplica.                                              | Estar + preposiciones + nombre expresan estado o situación temporal; estar para + infinitivo para inminencia, valoraciones o estados; estar por para intención, duda o acción no realizada. |
| Ejemplos destacados      | Soy soltera, Alberto es alto, esta falda es de cuero.   | Estoy resfriado, está frío, estoy soltera (temporal), está roto, estamos asombrados.                |

:pencil: Encuentra más información sobre el uso de __ser__ y __estar__ y ejercicios en la [Guía de gramática - Ser y Estar](https://www.dropbox.com/scl/fi/vr9mkz1g2420quc2pjnjf/Gramatica_Ser_y_Estar.pdf?rlkey=dy70g7cpu3cvb0dy8tt2dv1l4&dl=0)

### Tiempos del pasado: Pretérito imperfecto e indefinido

Aqui tienes una tabla que resume el contraste en el uso del pretérito imperfecto y el pretérito indefinido.

| Uso principal                                          | Pretérito imperfecto                                                        | Pretérito indefinido                                            |
|--------------------------------------------------------|----------------------------------------------------------------------------|-----------------------------------------------------------------|
| Acción en desarrollo y acción puntual                  | Describe la acción en desarrollo o el contexto.                | Describe una acción puntual que ocurre durante la acción en desarrollo. |
| Acción puntual que interrumpe desarrollo               | No se usa para la acción puntual.                              | Se usa para la acción puntual que interrumpe la acción en desarrollo.   |
| Dos acciones en desarrollo al mismo tiempo             | Se usa para ambas acciones.                                    | No se usa para dos acciones en desarrollo.           |
| Narrar circunstancias y entorno                       | Se usa para las circunstancias que rodean el acontecimiento.   | Se usa para el acontecimiento central de la narración. |
| En narración: circunstancias, acciones y desarrollo    | Se usa para circunstancias o acciones en desarrollo.           | Se usa para acciones puntuales sucesivas.            |
| Usar estar + gerundio                                 | Se usa para describir acciones en desarrollo con algunos verbos. | Cambia el significado del verbo en algunos casos.     |
| Referencia temporal / duración                        | Se usa para la acción o situación en desarrollo durante el periodo indicado. | Se usa para la acción que marca un punto de referencia temporal.          |
| Ejemplos destacados                                   | Cuando estudiaba alemán, fui dos veces a Alemania. Veíamos la tele mientras comíamos. Mi primer hijo nació el 13 de enero de 2007. Estaba todo nevado y hacía un frío tremendo. | A Juan lo atacaron cuando paseaba por el parque. De repente oímos un grito. Me levanté, me puse la bata y bajé a mirar. |

:pencil: y algunas actividades adicionales sobre el uso de los dos tiempos en la [Guía de gramática - Uso de imperfecto e indefinido](https://www.dropbox.com/scl/fi/jbu2jorsf0g5i94dde266/Gramatica_Imperfecto_Indefinido_II.pdf?rlkey=xuvf648hcrxf8lhyii347n4vn&dl=0)
